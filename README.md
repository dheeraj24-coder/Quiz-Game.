# Quiz-Game.

🧠 Computer Quiz Game (Python)

A simple command-line based quiz game built using Python.
This project asks users basic questions about AI and ChatGPT and gives instant feedback.

🚀 Features
Interactive CLI (Command Line Interface)
Basic user input handling
Instant correct/incorrect feedback
Simple score system
Beginner-friendly Python project
📌 How It Works
The program welcomes the user.
Asks if the user wants to play.
If the user types "yes", the quiz starts.
Two questions are asked:
Full form of ChatGPT
Full form of AI
The program checks answers and displays results.
