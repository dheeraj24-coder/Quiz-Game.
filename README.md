# 🧠 Computer Quiz Game (Python)

A simple command-line based **Computer Quiz Game** built using Python.
This project asks users basic computer-related questions and calculates their score based on correct answers.

---

## 🚀 Features

* Interactive command-line quiz
* Multiple-choice style (user input based)
* Score tracking system
* Percentage calculation at the end
* Beginner-friendly Python project

---

## 🛠️ Technologies Used

* Python (Basic concepts)

  * `input()`
  * `if-else`
  * `string methods`
  * variables & arithmetic

---

## 📋 How It Works

1. The program greets the user.
2. Asks if the user wants to play.
3. If the user types `"yes"`, the quiz starts.
4. The user answers 4 computer-related questions:

   * CPU
   * GPU
   * RAM
   * PSU
5. The program:

   * Checks answers (case-insensitive)
   * Increments score for correct answers
6. At the end, it displays:

   * Total correct answers
   * Percentage score

---

## ▶️ How to Run

1. Make sure Python is installed on your system.
2. Copy the code into a file named:

```bash
quiz.py
```

3. Run the program:

```bash
python quiz.py
```
